# GlobalTech-Solutions-Management-Solutions

📌 Overview

This project demonstrates how to use Python dictionaries to manage and analyze customer and project data. It simulates a simple customer management system for a company offering various technical services.

The program stores customer details, tracks projects, calculates costs, and generates useful business insights.

⚙️ Technologies Used
Python
Dictionaries
Lists
Basic Data Structures
📁 Features
1. Service Management
Stores service categories and hourly rates
Includes services such as:
Web Development
Data Analysis
Cybersecurity
AI Development
2. Customer Data Management
Creates individual customer records
Stores:
Company name
Contact person
Email
Phone number
Combines all customers into a master dictionary
3. Customer Lookup & Updates
Retrieves specific customer information
Handles missing customers safely using .get()
Updates customer details dynamically
4. Project Tracking
Assigns projects to customers
Stores:
Project name
Service type
Hours worked
Budget
Status (active, completed, pending)
5. Cost Calculation
Calculates project cost using:
cost = hourly rate × hours
Adds cost directly to each project
6. Data Analysis
Counts service usage across all projects
Calculates:
Total hours
Total budget
Average project budget
Highest and lowest budgets
7. Customer Insights
Lists all customers and companies
Identifies active customers
Calculates total budget per customer
Generates customer summary reports
8. Advanced Features
Adjusts service rates using dictionary comprehension
Categorizes services into pricing tiers
Validates customer data structure
Tracks project statuses (active, completed, pending)
Performs detailed budget analysis per customer
▶️ How to Run
Make sure Python is installed
Run the script:
python module08-assignment.py
